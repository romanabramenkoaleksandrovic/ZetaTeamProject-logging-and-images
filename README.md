# Проект: Стеганографическое средство сокрытия в метаданных изобранжения
**Команда разработки: ZetaEngineering**

В данном проекте реализован LSB метод сокрытия текста в изображении на языке C# с использованием Avalonia UI.

<details>
<summary>Посмотреть детальное описание LSB</summary>

• Метод наименее значимого бита (LSB). Замена наименее значимого бита в каждом канале цвета на бит скрываемого сообщения. Поскольку изменение последнего бита незначительно влияет на общий цвет пикселя, человеческий глаз не замечает разницы. Однако этот метод чувствителен к сжатию с потерями и другим видам обработки изображений, которые могут разрушить скрытую информацию.
</details>

## Принцип работы метода
1. **Шифровка сообщения в изображении.**  
&emsp;1.1. Получение файла изображения.    
&emsp;&emsp;1.1.1. С помощью поля "Выберите файл:" и кнопки "Выбарть";  
&emsp;&emsp;1.1.2. С помощью реализованной системы Grag-and-drop.  
&emsp;1.2. Получение текста из поля "Текст:".  
&emsp;1.3. Разбиение полученного текста на биты, в конце добавляем бит-терминатор ("\0").  
&emsp;1.4. Проверка сможем ли вместить полученный массив битов текста в изображение.  
&emsp;1.5. Прогон с 0 пикселя: в каждом канале каждого пикселя меняем наименее значимый бит на бит нашего текста, пока не закончится полученный массив битов текста.  
&emsp;1.6. Сохраняем изменённое изображение в формате PNG.  

2. **Расшифровка сообщения из изображения.**  
&emsp;2.1. Получение файла изображения.  
&emsp;&emsp;2.1.1. С помощью поля "Выберите файл:" и кнопки "Выбарть";  
&emsp;&emsp;2.1.2. С помощью реализованной системы Grag-and-drop.  
&emsp;2.2. Прогон с 0 пикселя: с 0 пикселя собираем наименее значимые биты в массив, пока не наткнёмся на бит-терминатор ("\0").  
&emsp;2.3. Конвертация полученного массива битов в текст.  
&emsp;2.4. Отображение расшифрованного текста в поле "Текст:".


## Используемые библиотеки библиотеки:  
**• Avalonia UI** - для красивой визуальной части и кроссплатформенности приложения.  
**• Avalonia.Media.Imaging** - для отрисовки графики и попиксельного изменения изображения.  
**• Avalonia.Platform.Storage** - для вызова диалоговых окон выбора изображения и сохранения.  
**• System.Collections.Generic** - для динамического хранения битовых массивов текста.  
**• System.IO** - для работы с файловой системой.  
**• System.Linq** - доп. библиотека для работы с файлами.  
**• System.Threading.Tasks** - для реализации асинхронных методов.  
**• SixLabors.ImageSharp** - для загрузки, изменения и сохранения изображений.  
**• SixLabors.ImageSharp.Formats.Png** - для точного декодирования и кодирования формата PNG.  
**• SixLabors.ImageSharp.PixelFormats** - для доступа к каналам пикселе (RGBA32).  
**• System.Text** - для преобразование текста в набор битов (UTF8) и обратно.
